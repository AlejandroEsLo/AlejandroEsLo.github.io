VIRTUAL REALITY COMMUNICATIONS TRACES SIMULATOR

SUMMARY

This project presents a tool for the construction of a 3D network simulator and Virtual Reality in which a packet exchange takes place.

Its function is to read the capture that we introduce and interpret the fields that interest us, to be able to represent the complete network that is reflected in the capture and visualize it in a Virtual Reality scene.

The project is based on A-Frame technology, a JavaScript framework, which is used to create virtual reality scenes.
